# Docs folder

This folder contains project documentation and shared assets.

Key files:

- `dataset_stats.md`: train/val/test stats for TinyStories splits
- `github_board.md`: suggested GitHub Project board setup
- `human_rubric.md`: human evaluation rubric (W4)
- `human_eval_template.csv`: scoring template for manual evaluation
- `sample_outputs.md`: baseline vs tuned examples
- `team_roles.md`: roles, branches, and weekly tasks
- `source_project_brief.pdf`, `source_day1_summary.pdf`: course materials

Assets:

- `assets/`: images used in docs or reports
